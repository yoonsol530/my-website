# my-website

1. 디자인 컨셉: Glassmorphism (유리 질감)이 대시보드의 가장 큰 특징은 시각적인 미니멀리즘과 현대적인 UI입니다.배경 디자인: radial-gradient를 사용하여 파스텔 톤의 부드러운 배경을 만들었습니다.유리 효과: 카드 섹션에 backdrop-filter: blur(20px)를 적용하여, 뒷배경이 은은하게 비치는 반투명 유리 같은 고급스러운 느낌을 줍니다.애니메이션: 페이지를 새로고침할 때 내용이 아래에서 위로 살짝 올라오며 나타나는 fadeIn 효과가 적용되어 있어 사용자 경험(UX)이 매끄럽습니다.

2. 주요 기능 블록 설명
① 헤더 (상단 영역)날짜: new Date().toLocaleDateString('de-DE', ...)를 사용하여 접속한 날짜를 독일어 형식(예: Montag, 4. Mai 2026)으로 자동 변환해 보여줍니다.
명언: 독일의 문호 괴테나 철학자 비트겐슈타인의 명언 중 하나가 무작위(Random)로 표시되어 학습 동기를 부여합니다.
② 통합 검색 시스템단순한 검색창이 아니라 검색 엔진 전환 기능이 핵심입니다.
Google: 일반적인 정보 검색dict.cc: 독일어-영어/한국어 단어 사전 검색Duden: 독일인의 국어사전격인 '두덴' 검색사용자가 선택한 버튼에 따라 입력한 키워드를 해당 사이트의 검색 결과 페이지로 바로 연결해 줍니다.
③ 큐레이션된 학습 리소스학습 효율을 높이기 위해 리소스를 세 가지 카테고리로 분류해 두었습니다.
HÖREN & SEHEN (듣기/보기): 뉴스(Tagesschau), DW 교육 콘텐츠 등 시청각 자료.
LESEN (읽기): Die Zeit, Der Spiegel 등 독일의 주요 정론지.WERKZEUGE (도구): 번역기(DeepL), 사전 등 보조 도구.
④ 로컬 메모장 (Notizen)textarea에 입력한 내용은 브라우저의 localStorage에 저장됩니다.이 기능 덕분에 페이지를 새로고침하거나 브라우저를 껐다 켜도 적어둔 단어나 메모가 사라지지 않고 그대로 유지됩니다.

3. 코드의 기술적 특징 (Code Stack)기술적용된 부분Lucide Iconsdata-lucide 속성을 통해 헤드폰, 책, 돋보기 등의 아이콘을 렌더링함.CSS Variables:root에 색상과 그림자 값을 변수로 정의하여 디자인 통일성을 유지함.Event Listeners엔터 키 입력 시 검색 실행(handleSearch), 메모 입력 시 자동 저장 기능.Responsivegrid-template-columns: repeat(auto-fit, ...)로 모바일과 PC 화면에 모두 대응함.


// 메모 자동 저장 기능
memoArea.addEventListener('input', (e) => {
    localStorage.setItem(STORAGE_KEY, e.target.value);
});
검색 로직: 사용자가 선택한 엔진에 맞춰 URL 쿼리 파라미터(?q=, ?s=)를 조합하여 새 탭에서 결과를 열어줍니다.

로컬 스토리지: 서버가 없어도 사용자의 브라우저 데이터베이스에 메모 내용을 저장하므로 개인 학습용 대시보드로 적합합니다.
